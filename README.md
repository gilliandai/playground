# Playground

这个仓库用于发布一些可以直接在浏览器里运行的小工具。

## Tools

- [阅读书签](./reading-bookmark/)

## 隐私边界

- 阅读书签首次打开是空书库。
- 仓库不包含内测数据库。
- 阅读书签不包含预置分类选项或预置标签。
- 书库数据只保存在访问者自己的浏览器本地。
- GitHub Pages 只托管网页文件，不会保存用户的书库数据。
