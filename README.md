# 阅读书签 Public Deploy

这是适合发布到 GitHub Pages 的公开版。

## 隐私边界

- 首次打开是空书库。
- 不包含内测数据库。
- 不包含预置分类选项或预置标签。
- 书库数据只保存在访问者自己的浏览器本地。
- GitHub Pages 只托管网页文件，不会保存用户的书库数据。

## GitHub Pages

把这个文件夹里的内容提交到用于发布的分支或目录，然后在仓库 Settings -> Pages 里选择发布源即可。

如果使用项目站点，访问地址通常类似：

```text
https://你的用户名.github.io/仓库名/
```

## 同步

不同设备之间仍然通过“导出同步文本”与“导入同步文本”手工同步。
